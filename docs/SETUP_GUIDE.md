## Prerequisites
- n8n installed
- Airtable account
- OpenRouter API key

## Installation Steps
1. Import the `maritime-supply-chain-monitor.json` into n8n
2. Configure credentials (Airtable, OpenRouter, SMTP)
3. Update API keys in nodes
4. Test the workflow
5. Activate schedule trigger
